## Engineering themes

Use these as real connective tissue, not decorative values:

1. **Judgment in models, invariants in code.** Models decide content; code controls authority,
   state, limits, visibility, and transactions.
2. **Evidence before confidence.** Real-provider smokes, repeat runs, debug JSONL, replay, negative
   controls, and documented acceptance artifacts.
3. **Interfaces for difficult workflows.** Turn opaque prompt batches, multi-agent calls, or nested
   document generation into inspectable operations.
4. **Forward-only contracts.** Prefer explicit current contracts over long-lived compatibility
   layers in young systems.
5. **Local-first where the data is personal.** SceneQueue and Alex Tavern keep local runtime data
   outside Git.
